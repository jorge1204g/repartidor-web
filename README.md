# Aplicación Web de Repartidor

Aplicación web para que los repartidores gestionen sus pedidos, historial, mensajes y perfil.

## Características

- Gestión de pedidos activos
- Historial de entregas
- Sistema de mensajería con el administrador
- Perfil del repartidor
- Autenticación por ID

## Tecnologías utilizadas

- React
- TypeScript
- Firebase
- Vite
- Material UI

## Instalación

1. Clona el repositorio
2. Ejecuta `npm install`
3. Ejecuta `npm run dev`

## Despliegue

Esta aplicación está configurada para ser desplegada fácilmente en plataformas como Vercel o Netlify.
